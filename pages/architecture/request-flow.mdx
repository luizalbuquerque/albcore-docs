# Fluxo de Requisição

Veja como uma requisição percorre todas as camadas do AlbCore:

## Exemplo: Criar Agendamento

```
POST /api/v1/schedules
Headers: X-Tenant-Id: tenant-abc
         Authorization: Bearer eyJ...
Body:    { "title": "Daily sync", "executionTime": "2026-06-01T10:00" }
```

### Passo a Passo

```
1. TenantInterceptor.preHandle()
   ├── Lê header X-Tenant-Id: "tenant-abc"
   └── TenantContext.setTenantId("tenant-abc")

2. JwtAuthenticationFilter.doFilterInternal()
   ├── Extrai Bearer token
   ├── JwtProvider.validateToken(token) → true
   ├── JwtProvider.extractSubject(token) → "user-123"
   └── Popula SecurityContext com Authentication

3. ScheduleController.create(@Valid @RequestBody ScheduleRequest)
   ├── Bean Validation (@NotBlank, @Future, etc.)
   └── Chama service.create(request)

4. ScheduleApplicationService.create(request)
   ├── Cria ScheduleEntity com tenantId = "tenant-abc"
   ├── entity.validate() → regras de domínio
   └── repository.save(entity) → retorna ScheduleResponse

5. ScheduleRepositoryAdapter.save(entity)
   ├── toDomainEntity(entity) → ScheduleJpa
   └── springRepository.save(jpaEntity) → ScheduleJpa salvo

6. Database (PostgreSQL)
   └── INSERT INTO schedule (id, tenant_id, title, ...) VALUES (...)

7. Resposta 201 Created
   └── ApiResponse.success(scheduleResponse)
```
