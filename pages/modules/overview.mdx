# Módulos do AlbCore

Os módulos são **blocos funcionais reutilizáveis** ativados por **Feature Flag** por tenant.
Cada produto white label escolhe quais módulos ativar.

## Como Ativar um Módulo

```java
// Verificação de feature flag no service
@Service
public class MarketplaceApplicationService {

    @Autowired
    private FeatureFlagService featureFlags;

    @Autowired
    private TenantContext tenantContext;

    public MarketplaceResponse create(...) {
        String tenantId = tenantContext.getTenantId();

        if (!featureFlags.isEnabled("MODULE_MARKETPLACE", tenantId)) {
            throw new ForbiddenException("Módulo Marketplace não ativo neste plano");
        }
        // ...
    }
}
```
