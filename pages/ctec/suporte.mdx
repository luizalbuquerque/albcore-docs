## Rede de Apoio à Integração

A Albuquerque Tech oferece uma rede completa de apoio para clientes CTEC que desejam
integrar a plataforma de rastreamento em seus sistemas.

### Níveis de Suporte

| Nível | Incluso | SLA |
|-------|---------|-----|
| **Básico** | Documentação + FAQ + e-mail | Resposta em até 48h |
| **Profissional** | Consultoria técnica + call de integração | Resposta em até 24h |
| **Enterprise** | Engenheiro dedicado + implementação assistida | Resposta em até 4h |

### O Que Oferecemos

- **Consultoria de Integração** — Chamadas técnicas para alinhar arquitetura e fluxo de dados
- **Ambiente de Sandbox** — Servidor de testes com dados simulados para desenvolvimento

- **Monitoramento Proativo** — Alertas sobre indisponibilidade da API ou mudanças de versão

## FAQ — Perguntas Frequentes

### A sessão (jsession) expira?

Sim, após ~30 minutos de inatividade. Ao receber `result: 5`, refaça o login.

### Quantas requisições posso fazer por minuto?

Não há rate limit documentado, mas recomendamos no máximo **60 requisições/minuto** por sessão.
Para monitoramento de frota grande (+100 veículos), use batch requests com múltiplos `devIdno` separados por vírgula.

### O vídeo funciona em HTTPS?

O servidor de mídia atualmente opera em HTTP. Para HTTPS, é necessário configurar um proxy reverso (nginx/Traefik) com certificado SSL.

### Posso usar WebSocket para dados em tempo real?

A API atual é baseada em polling (HTTP). WebSocket não está disponível nativamente, mas é possível implementar um gateway intermediário que faz polling e publica via WS.

### Como converter as coordenadas para usar no Google Maps?

Passe `toMap=1` na requisição. Sem isso, as coordenadas vêm em WGS84 — divida por 1.000.000 para obter graus decimais compatíveis com OpenStreetMap/Mapbox.
