# Veículos & GPS

Endpoints para consultar veículos autorizados, status GPS em tempo real, posição, velocidade e dispositivos associados.

---

## Listar Veículos do Usuário

Retorna todos os veículos e dispositivos autorizados para o usuário logado.

<div style={{display:'flex',gap:'0.5rem',alignItems:'center',margin:'0.5rem 0'}}>
  <span className="method-badge method-get">GET</span>
  <span className="method-badge method-post">POST</span>
  <code className="endpoint-url">/StandardApiAction_queryUserVehicle.action</code>
</div>

### Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:-----------:|-----------|
| `jsession` | `string` | Sim | Token de sessão |
| `language` | `string` | Não | Idioma (`zh` ou `en`) |

### Resposta — Veículo

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `id` | `number` | ID do veículo |
| `nm` | `string` | Placa / identificador |
| `ic` | `number` | Ícone do veículo |
| `pid` | `number` | ID da empresa/equipe |
| `pnm` | `string` | Nome da empresa |
| `dl` | `array` | Lista de dispositivos (ver abaixo) |
| `vehicleType` | `number` | Tipo: 1=carga perigosa, 2=passageiro, 4=táxi, 5=carga |

### Resposta — Dispositivo (`dl[]`)

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `id` | `string` | Número do dispositivo (DevIdno) |
| `cc` | `number` | Quantidade de canais de vídeo |
| `cn` | `string` | Nomes dos canais (separados por `,`) |
| `tc` | `number` | Quantidade de sensores de temperatura |
| `md` | `number` | Módulos suportados (bitwise) |
| `sim` | `string` | Número do SIM card |
| `payEnd` | `number` | Expiração do serviço (timestamp) |

### Exemplo

<Tabs items={['cURL', 'Python', 'JavaScript', 'Java', 'C#']}>

<Tabs.Tab>
```bash
curl "http://39.100.229.38:8088/StandardApiAction_queryUserVehicle.action?jsession=SEU_JSESSION"
```
</Tabs.Tab>

<Tabs.Tab>
```python
import requests

response = requests.get(
    f"{BASE_URL}/StandardApiAction_queryUserVehicle.action",
    params={"jsession": jsession}
)
data = response.json()

for vehicle in data.get("vehicles", []):
    print(f"Placa: {vehicle['nm']}, ID: {vehicle['id']}")
    for dev in vehicle.get("dl", []):
        print(f"  Dispositivo: {dev['id']}, Canais: {dev.get('cc', 0)}")
```
</Tabs.Tab>

## Obter Dispositivo pela Placa

Retorna o(s) dispositivo(s) associado(s) a uma ou mais placas.

<div style={{display:'flex',gap:'0.5rem',alignItems:'center',margin:'0.5rem 0'}}>
  <span className="method-badge method-get">GET</span>
  <span className="method-badge method-post">POST</span>
  <code className="endpoint-url">/StandardApiAction_getDeviceByVehicle.action</code>
</div>

### Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:-----------:|-----------|
| `jsession` | `string` | Sim | Token de sessão |
| `vehiIdno` | `string` | Não | Placa(s), separadas por `,` |

### Exemplo

<Tabs items={['cURL', 'Python', 'JavaScript', 'Java', 'C#']}>

<Tabs.Tab>
```bash
curl "http://39.100.229.38:8088/StandardApiAction_getDeviceByVehicle.action?jsession=SEU_JSESSION&vehiIdno=ABC-1234"
```
</Tabs.Tab>

<Tabs.Tab>
```python
response = requests.get(
    f"{BASE_URL}/StandardApiAction_getDeviceByVehicle.action",
    params={"jsession": jsession, "vehiIdno": "ABC-1234"}
)

for d in devices:
    tipo = "Vídeo" if d["type"] == 1 else "GPS"
    print(f"Dispositivo: {d['did']} ({tipo})")
```
</Tabs.Tab>

</Tabs.Tab>

<Tabs.Tab>
```java

        + "?jsession=" + jsession + "&vehiIdno=ABC-1234"))
    .GET().build();

HttpResponse<String> response = client.send(request,
    HttpResponse.BodyHandlers.ofString());

    .getAsJsonObject().getAsJsonArray("devices");
```

</Tabs.Tab>

<Tabs.Tab>
```csharp

    $"{BaseUrl}/StandardApiAction_getDeviceByVehicle.action"
    + $"?jsession={jsession}&vehiIdno=ABC-1234");

foreach (var dev in json.GetProperty("devices").EnumerateArray())
{
    var tipo = dev.GetProperty("type").GetInt32() == 1 ? "Vídeo" : "GPS";

```
</Tabs.Tab>

</Tabs>

```json
  ]

```

| Campo | Descrição |
|-------|-----------|
| `type=1` | Dispositivo de vídeo |

## Status Online do Dispositivo

Verifica se o(s) dispositivo(s) estão online ou offline.

<div style={{display:'flex',gap:'0.5rem',alignItems:'center',margin:'0.5rem 0'}}>
  <span className="method-badge method-get">GET</span>
  <span className="method-badge method-post">POST</span>
  <code className="endpoint-url">/StandardApiAction_getDeviceOlStatus.action</code>
</div>

### Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:-----------:|-----------|
| `jsession` | `string` | Sim | Token de sessão |
| `devIdno` | `string` | Não | Número(s) do dispositivo, separados por `,` |
| `vehiIdno` | `string` | Não | Placa(s) — usado se `devIdno` estiver vazio |
| `status` | `number` | Não | Filtrar: `0`=offline, `1`=online, vazio=todos |

## Status GPS (Posição Atual)

Retorna a posição GPS mais recente, velocidade, direção, status de sensores e mais.

<div style={{display:'flex',gap:'0.5rem',alignItems:'center',margin:'0.5rem 0'}}>
  <span className="method-badge method-get">GET</span>
  <span className="method-badge method-post">POST</span>
  <code className="endpoint-url">/StandardApiAction_getDeviceStatus.action</code>
</div>

### Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:-----------:|-----------|
| `jsession` | `string` | Sim | Token de sessão |
| `devIdno` | `string` | Não | Número(s) do dispositivo |
| `vehiIdno` | `string` | Não | Placa(s) — fallback se `devIdno` vazio |
| `geoaddress` | `number` | Não | `1` = resolver endereço geográfico |
| `toMap` | `number` | Não | `1`=Google Maps coords, `2`=Baidu coords |
| `language` | `string` | Não | `zh` para endereço em chinês |
| `driver` | `number` | Não | `1` = incluir info do motorista |

### Campos da Resposta

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `id` | `string` | DevIdno |
| `lng` | `number` | Longitude (÷ 1.000.000 para graus). Ex: `113921858` → `113.921858` |
| `lat` | `number` | Latitude (÷ 1.000.000 para graus) |
| `sp` | `number` | Velocidade (÷ 10 para km/h). Ex: `520` → `52 km/h` |
| `ol` | `number` | Online: `1`=online, `0`=offline |
| `gt` | `string` | Hora do último GPS (`yyyy-MM-dd HH:mm:ss`) |
| `hx` | `number` | Direção (0°=Norte, sentido horário, max 360°) |
| `mlng` | `string` | Longitude convertida para o mapa |
| `mlat` | `string` | Latitude convertida para o mapa |
| `pk` | `number` | Tempo estacionado (segundos) |
| `lc` | `number` | Quilometragem total (metros) |
| `yl` | `number` | Combustível (÷ 100 para litros) |
| `ps` | `string` | Endereço resolvido (quando `geoaddress=1`) |
| `t1`–`t4` | `number` | Sensores de temperatura |
| `dn` | `string` | Nome do motorista |
| `net` | `number` | Rede: 0=3G, 1=WiFi, 2=cabeada, 3=4G, 4=5G |

<Callout type="warning">
**Conversão de coordenadas:** `lng` e `lat` são inteiros. Divida por **1.000.000** para obter graus decimais. Use `mlng`/`mlat` se precisar de coordenadas já convertidas para o provedor de mapa (Google/Baidu).
</Callout>

### Exemplo

<Tabs items={['cURL', 'Python', 'JavaScript', 'Java', 'C#']}>

<Tabs.Tab>
```bash
curl "http://39.100.229.38:8088/StandardApiAction_getDeviceStatus.action?jsession=SEU_JSESSION&devIdno=500000&toMap=1"
```
</Tabs.Tab>

<Tabs.Tab>
```python
response = requests.get(
    f"{BASE_URL}/StandardApiAction_getDeviceStatus.action",
    params={
        "jsession": jsession,
        "devIdno": "500000",
        "toMap": 1,
        "geoaddress": 1
    }
)

    lng = device["lng"] / 1_000_000
    speed = device["sp"] / 10
    online = "Online" if device["ol"] == 1 else "Offline"
    print(f"GPS: {lat:.6f}, {lng:.6f} | {speed} km/h | {online}")
```
</Tabs.Tab>

```java
    double lng = d.get("lng").getAsDouble() / 1_000_000;
    double speed = d.get("sp").getAsDouble() / 10;

}
```
