# Alarmes

Consulta alarmes históricos e monitora alarmes em tempo real da frota.

---

## Alarmes Históricos (Paginado)

Retorna alarmes de um dispositivo/veículo em um período.

<div style={{display:'flex',gap:'0.5rem',alignItems:'center',margin:'0.5rem 0'}}>
  <span className="method-badge method-get">GET</span>
  <span className="method-badge method-post">POST</span>
  <code className="endpoint-url">/StandardApiAction_queryAlarmDetail.action</code>
</div>

### Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:-----------:|-----------|
| `jsession` | `string` | Sim | Token de sessão |
| `devIdno` | `string` | Não | Dispositivo(s), separados por `,` |
| `vehiIdno` | `string` | Não | Placa(s) — fallback se `devIdno` vazio |
| `begintime` | `string` | Sim | Início (`yyyy-MM-dd HH:mm:ss`) |
| `endtime` | `string` | Sim | Fim (máx. 90 dias de intervalo) |
| `armType` | `string` | Sim | Tipo(s) de alarme, separados por `,` |
| `handle` | `number` | Não | `0`=não tratado, `1`=tratado, vazio=todos |
| `currentPage` | `number` | Não | Página (default: 1) |
| `pageRecords` | `number` | Não | Registros por página (default: 10) |
| `toMap` | `number` | Não | `1`=Google, `2`=Baidu |
| `geoaddress` | `number` | Não | `1` = resolver endereço |

### Campos da Resposta

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `atp` | `number` | Tipo do alarme |
| `did` | `string` | Dispositivo |
| `vid` | `string` | Placa |
| `stm` | `number` | Timestamp início (UTC ms) |
| `etm` | `number` | Timestamp fim (UTC ms) |
| `guid` | `string` | ID único do alarme |
| `hd` | `number` | `0`=não tratado, `1`=tratado |
| `hdu` | `string` | Usuário que tratou |
| `hdc` | `string` | Conteúdo do tratamento |
| `img` | `string` | URLs de imagens (separadas por `;`) |
| `slng`/`slat` | `number` | Coordenadas início |
| `elng`/`elat` | `number` | Coordenadas fim |

### Exemplo

<Tabs items={['cURL', 'Resposta']}>

<Tabs.Tab>
```bash

jsession=SEU_JSESSION&devIdno=500000&\

armType=2,9,11&currentPage=1&pageRecords=50"
```

## Alarmes em Tempo Real

Retorna os alarmes ativos de um dispositivo **online**.

<div style={{display:'flex',gap:'0.5rem',alignItems:'center',margin:'0.5rem 0'}}>
  <span className="method-badge method-get">GET</span>
  <span className="method-badge method-post">POST</span>
  <code className="endpoint-url">/StandardApiAction_vehicleAlarm.action</code>
</div>

### Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:-----------:|-----------|
| `jsession` | `string` | Sim | Token de sessão |
| `DevIDNO` | `string` | Não | Número do dispositivo |
| `toMap` | `number` | Não | Conversão de coordenadas |

## Tipos de Alarme

| Código | Descrição |
|:------:|-----------|
| 2 | Excesso de velocidade |
| 9 | Entrada em área proibida |
| 11 | Saída de área (geofence) |
| 14 | Fadiga ao volante |
| 15 | Colisão |
| 16 | Capotamento |
| 17 | Porta aberta em movimento |
| 20 | Corte de combustível |
| 22 | Pânico (SOS) |
| 23 | Bateria fraca |
| 48 | Alarme ADAS (colisão frontal) |
| 49 | Desvio de faixa (ADAS) |
| 50 | Fadiga (DSM) |
| 51 | Fumando (DSM) |
| 52 | Celular ao volante (DSM) |

<Callout type="info">
Use o campo `armType` na consulta para filtrar por um ou mais tipos. Separe múltiplos tipos com vírgula: `armType=2,9,11`.
</Callout>
