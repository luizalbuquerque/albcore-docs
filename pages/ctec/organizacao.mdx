# Organização, Roles & Contas

Gerenciamento hierárquico de empresas, perfis de acesso e contas de usuário.

---

## Empresas (Organizações)

### Criar / Editar Empresa

| | |
|---|---|
| **URL** | `/StandardApiAction_addCompany.action` |
| **Método** | POST |

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `jsession` | string | Sim | Token |
| `companyName` | string | Sim | Nome da empresa |
| `parentId` | number | Não | ID da empresa-pai (hierarquia) |

### Consultar Empresa

| | |
|---|---|
| **URL** | `/StandardApiAction_findCompany.action` |
| **Método** | GET / POST |

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `jsession` | string | Sim | Token |
| `companyId` | number | Não | ID da empresa (vazio = todas) |

### Excluir Empresa

| | |
|---|---|
| **URL** | `/StandardApiAction_deleteCompany.action` |
| **Método** | POST |

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `jsession` | string | Sim | Token |
| `companyId` | number | Sim | ID da empresa |

## Contas de Usuário

### Criar / Editar Conta

| | |
|---|---|
| **URL** | `/StandardApiAction_addAccount.action` |
| **Método** | POST |

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `jsession` | string | Sim | Token |
| `account` | string | Sim | Nome de login |
| `password` | string | Sim | Senha |
| `companyId` | number | Sim | Empresa vinculada |
