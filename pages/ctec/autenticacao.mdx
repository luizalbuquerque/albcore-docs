# Autenticação

O sistema de autenticação utiliza sessões baseadas em token (`jsession`).
Após o login, o `jsession` deve ser enviado como query parameter em **todas** as requisições subsequentes.

<Callout type="info">
O token `jsession` tem validade limitada. Se receber `result: 5` em qualquer endpoint, faça login novamente.
</Callout>

---

## Login

Autentica o usuário e retorna um `jsession` válido.

<div style={{display:'flex',gap:'0.5rem',alignItems:'center',margin:'0.5rem 0'}}>
  <span className="method-badge method-get">GET</span>
  <span className="method-badge method-post">POST</span>
  <code className="endpoint-url">/StandardApiAction_login.action</code>
</div>

### Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:-----------:|-----------|
| `account` | `string` | Sim | Nome de usuário |
| `password` | `string` | Sim | Senha |

### Exemplo

<Tabs items={['cURL', 'Python', 'JavaScript', 'Java', 'C#']}>

<Tabs.Tab>
```bash
curl -X GET "http://39.100.229.38:8088/StandardApiAction_login.action?account=SEU_USER&password=SUA_SENHA"
```
</Tabs.Tab>

<Tabs.Tab>
```python
import requests

BASE_URL = "http://39.100.229.38:8088"

response = requests.get(
    f"{BASE_URL}/StandardApiAction_login.action",
    params={"account": "SEU_USER", "password": "SUA_SENHA"}
)
data = response.json()

if data["result"] == 0:
    jsession = data["jsession"]
    print(f"Autenticado! Token: {jsession}")
else:
    print(f"Erro no login: código {data['result']}")
```

<Tabs.Tab>
```java
import java.net.http.*;
import java.net.URI;
import com.google.gson.*;

public class CTECAuth {
    static final String BASE_URL = "http://39.100.229.38:8088";

    public static String login(String account, String password) throws Exception {
        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create(BASE_URL + "/StandardApiAction_login.action"
                + "?account=" + account + "&password=" + password))
            .GET()
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());

        JsonObject json = JsonParser.parseString(response.body()).getAsJsonObject();
        if (json.get("result").getAsInt() == 0) {
            return json.get("jsession").getAsString();
        }
        throw new RuntimeException("Login falhou: " + json.get("result"));
    }
}
```
</Tabs.Tab>

<Tabs.Tab>
```csharp
using System.Net.Http;
using System.Text.Json;

public class CTECAuth
{
    private const string BaseUrl = "http://39.100.229.38:8088";
    private static readonly HttpClient _http = new();

    public static async Task<string> LoginAsync(string account, string password)
    {
        var url = $"{BaseUrl}/StandardApiAction_login.action"
                + $"?account={account}&password={password}";

        var response = await _http.GetStringAsync(url);
        var json = JsonDocument.Parse(response).RootElement;

        if (json.GetProperty("result").GetInt32() == 0)
            return json.GetProperty("jsession").GetString()!;

        throw new Exception($"Login falhou: {json.GetProperty("result")}");
    }
}
```

### Resposta de Sucesso

```json
{
  "result": 0,
  "jsession": "cf6b70a3-c82b-4392-8ab6-bbddce336222",
  "account_name": "ctec",
  "JSESSIONID": "cf6b70a3-c82b-4392-8ab6-bbddce336222"
}
```

| Campo | Tipo | Descrição |
|-------|------|-----------|
| `result` | `number` | `0` = sucesso |
| `jsession` | `string` | Token de sessão — usar em todas as chamadas seguintes |
| `account_name` | `string` | Nome da conta autenticada |

### Códigos de Erro

| result | Significado |
|:------:|-------------|
| `1` | Credenciais inválidas |
| `2` | Credenciais inválidas |
| `3` | Conta desabilitada |
| `4` | Conta expirada |
| `34` | SSO — conta já logada em outra sessão |

### Acesso ao Painel Web

Após login, é possível redirecionar o usuário para o painel web:

```
http://39.100.229.38:8088/808gps/index.html?userSession={jsession}&clientLogin=2
```

| Parâmetro | Descrição |
|-----------|-----------|
| `clientLogin=1` | Painel sem vídeo |
| `clientLogin=2` | Painel com vídeo e localização |
| `menuIds=25,26` | Menus específicos (25=vídeo, 26=playback) |

## Logout

Encerra a sessão do usuário e invalida o `jsession`.

<div style={{display:'flex',gap:'0.5rem',alignItems:'center',margin:'0.5rem 0'}}>
  <span className="method-badge method-get">GET</span>
  <span className="method-badge method-post">POST</span>
  <code className="endpoint-url">/StandardApiAction_logout.action</code>
</div>

### Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|:-----------:|-----------|
| `jsession` | `string` | Sim | Token de sessão |
