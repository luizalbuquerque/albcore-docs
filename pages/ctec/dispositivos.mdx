# Dispositivos

Gerenciamento de dispositivos GPS/vídeo e veículos na plataforma.

## Consultar Informações do Dispositivo

| | |
|---|---|
| **URL** | `/StandardApiAction_getDeviceInfo.action` |
| **Método** | GET / POST |

### Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `jsession` | string | Sim | Token |
| `devIdno` | string | Sim | Número do dispositivo |

---

## Adicionar Dispositivo

| | |
|---|---|
| **URL** | `/StandardApiAction_addDevice.action` |
| **Método** | POST |

### Parâmetros

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `jsession` | string | Sim | Token |
| `devIdno` | string | Sim | Número/IMEI do dispositivo |
| `companyId` | number | Sim | ID da empresa |
| `sim` | string | Não | Número do SIM card |
| `channelCount` | number | Não | Quantidade de canais de vídeo |

### Resposta

```json
{
  "result": 0
}
```

### Erros Comuns

| Código | Significado |
|--------|-------------|
| 14 | Limite de dispositivos atingido |
| 15 | Dispositivo já existe |
| 17 | Dispositivo já em uso |
